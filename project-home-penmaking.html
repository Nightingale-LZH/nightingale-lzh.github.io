Aeolia A1 CusX750

Piston-filling Conversion

This is a kit that converts Jinhao X750 to a piston-filling pen. It enables users to change the ink saturation in feed easily, especially when paired with Zebra-G flex-nib conversion.

Aeolia A2 CusM63

Rollerball

This is a rollerball pen with unibody design and optimizes for tripod grip. It uses Lamy M63 rollerball refill. The entire pen barrel is solid one-piece, hence unibody, without any separations. However, the friction-fitted pen tip retaining the refill often pops out under high writing pressure. This problem has been fixed in Aeolia A10.

Aeolia A3 CusVanishing

The Gold-Nib Fountain Pen

This is a fountain pen has the same classic silhouette as other Aeolia pens. It is based on the Pilot Vanishing Point inner pen, which has gold-nib option under $100 (at least in 2024). It has a gap-free smooth barrel with tripod grip indents.
